# Projeto: Lista de Tarefas (To-Do List)
- Este projeto implementa uma aplicação de lista de tarefas (To-Do List) que demonstra a manipulação dinâmica do DOM. Os usuários podem adicionar novas tarefas, marcar tarefas como concluídas, editar o texto da tarefa e deletá-las.

## Tecnologias utilizadas
- HTML para a estrutura básica da aplicação.

- CSS para estilização do layout e do estado das tarefas (concluídas).

- JavaScript para a lógica da aplicação, incluindo as operações de CRUD (Create, Read, Update, Delete) no DOM.

## O que aprendi
- Manipular o DOM para criar, adicionar e remover elementos dinamicamente (tarefas).

- Utilizar a Delegação de Eventos para gerenciar cliques em elementos criados dinamicamente (deletar e editar).

- Implementar a funcionalidade de edição de texto de um elemento, utilizando inputs ocultos e o evento keydown para confirmar (tecla Enter).

- Alterar o estilo de elementos (ex: adicionar a classe .done para tarefas concluídas) com base em eventos (checkbox).
