# Projeto: Login Responsivo
- Este projeto foca na responsividade de uma página de login. O layout original de duas colunas (formulário e imagem) é transformado em um layout de coluna única e adaptado para dispositivos móveis usando Media Queries, garantindo que o formulário seja utilizável em qualquer tamanho de tela.

## Tecnologias utilizadas
- HTML para a estrutura da página de login e do formulário.

- CSS para a estilização e a aplicação de regras de Media Queries para a quebra de layout em telas menores.

## O que aprendi
- Criar um layout responsivo complexo que altera radicalmente a estrutura da página em diferentes breakpoints.

- Utilizar Media Queries para esconder ou exibir elementos de design específicos (ex: esconder a imagem lateral em mobile).

- Reorganizar a ordem e o estilo de elementos do formulário (botões e links) para otimizar a experiência do usuário em telas menores.
