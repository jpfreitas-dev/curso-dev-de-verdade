# Projeto: Header Responsivo
- Este projeto implementa um cabeçalho de navegação (header) que se adapta a diferentes tamanhos de tela. O objetivo é praticar a responsividade, exibindo a navegação completa em telas grandes e um menu dropdown ativado por um botão em telas pequenas.

## Tecnologias utilizadas
- HTML para a estrutura do cabeçalho e da navegação.

- CSS para estilização e, principalmente, para o uso de Media Queries (@media) que controlam a adaptação do layout.

- JavaScript para a lógica de toggle (abrir/fechar) do menu dropdown em dispositivos móveis.

## O que aprendi
- Criar um layout de navegação responsivo usando Media Queries para alternar entre diferentes estilos e visibilidade de elementos.

- Implementar a funcionalidade de um menu dropdown (menu hambúrguer) usando JavaScript.

- Utilizar e gerenciar atributos de acessibilidade (ARIA) como aria-expanded e aria-hidden em elementos interativos.
