# Projeto: Jogo da Forca
- Este é a implementação do clássico Jogo da Forca. O projeto foca em desenvolver a lógica do jogo em JavaScript, manipulando a interface para exibir o progresso da forca e as letras acertadas. Inclui funcionalidades de pausa/reinício e notificação de vitória/derrota.

## Tecnologias utilizadas
- HTML para a estrutura da forca, letras e modais.

- CSS para a estilização do jogo, das letras e a criação dos modais (pause e mensagem final).

- JavaScript para a lógica do jogo (sorteio de palavras, contagem de vidas, verificação de letras) e manipulação do DOM.

## O que aprendi
- Implementar lógica de jogo em JavaScript, incluindo controle de estado, contagem de vidas e verificação de condições de vitória/derrota.

- Manipular o DOM para exibir partes do boneco da forca progressivamente, com base nas vidas restantes (usando a propriedade .classList.remove('hidden')).

- Criar e gerenciar a visibilidade de modais (caixas de diálogo) para interrupção do jogo e mensagens finais.

- Reiniciar o estado da aplicação para um novo jogo.
