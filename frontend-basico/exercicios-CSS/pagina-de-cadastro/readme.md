# Projeto: Página de Cadastro e Login
- Este projeto consiste em duas páginas (Cadastro e Login) com o objetivo de praticar layouts complexos utilizando CSS puro. O design é caracterizado por um layout dividido em duas colunas (formulário e imagem de destaque), estilização "bruta" com bordas grossas e a inclusão de elementos de login social.

## Tecnologias utilizadas
- HTML para a estrutura dos formulários e da página.

- CSS para estilização e layout, incluindo a organização em duas colunas.

## O que aprendi
- Estruturar páginas de formulário de login e cadastro.

- Utilizar display: grid para criar layouts de duas colunas (formulário e imagem).

- Estilizar formulários, inputs e botões, aplicando um design coeso.

- Adicionar imagens de fundo e de elementos (logo, ícones de redes sociais).
