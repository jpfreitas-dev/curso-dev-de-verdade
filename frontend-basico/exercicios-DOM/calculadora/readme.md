# Projeto: Calculadora Simples
- Este é um projeto de uma calculadora simples com operações aritméticas básicas (+, -, ×, ÷). O objetivo foi praticar a manipulação do DOM e a lógica de programação em JavaScript para processar a entrada do usuário e realizar os cálculos.

## Tecnologias utilizadas
- HTML para a estrutura da calculadora (display e botões).

- CSS para estilização, cores e o layout do teclado usando CSS Grid.

- JavaScript para a lógica de funcionamento, manipulação do DOM e realização dos cálculos (usando a função eval).

## O que aprendi
- Manipular o DOM para construir e atualizar a interface da calculadora (exibir valores no display e processar cliques dos botões).

- Implementar a lógica de entrada de dados e operações matemáticas.

- Gerenciar o estado da expressão e do display.

- Utilizar a função eval() para calcular o resultado de uma expressão.

- Realizar tratamento de erros simples (try...catch).
